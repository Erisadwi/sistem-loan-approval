# 💰 Sistem Loan Approval

Sistem Loan Approval merupakan aplikasi berbasis web yang dirancang untuk membantu proses pengajuan dan persetujuan pinjaman secara lebih efektif dan terstruktur. Sistem ini memungkinkan pengelolaan data pengajuan pinjaman serta mendukung proses evaluasi dan pengambilan keputusan terhadap setiap pengajuan yang diajukan.

## 📌 Fitur Utama

* Login dan Logout
* Dashboard Sistem
* Pengelolaan Data Pengajuan Pinjaman
* Tambah Data Pengajuan
* Edit Data Pengajuan
* Hapus Data Pengajuan
* Proses Persetujuan (Approval) Pinjaman
* Menampilkan Status Pengajuan
* Penyimpanan Data Menggunakan Database MySQL

## 🛠️ Teknologi yang Digunakan

* PHP
* MySQL
* HTML
* CSS
* Bootstrap
* JavaScript

## 📋 Modul Sistem

### Admin

* Login dan Logout
* Kelola Data Pengajuan Pinjaman
* Menambah Data Pengajuan
* Mengubah Data Pengajuan
* Menghapus Data Pengajuan
* Melakukan persetujuan atau penolakan pinjaman

### User

* Mengajukan Pinjaman
* Melihat Status Pengajuan
* Melihat Informasi Pinjaman

## 🎯 Tujuan Sistem

Sistem ini dikembangkan untuk:

* Mempermudah proses pengajuan pinjaman.
* Membantu proses verifikasi dan persetujuan pinjaman.
* Mengurangi pengelolaan data secara manual.
* Meningkatkan efisiensi dalam pengelolaan data pinjaman.
* Mendukung pengambilan keputusan secara lebih cepat dan terstruktur.
